Beosmapper project

With this application you can add building entrances to Open Street Maps to assist people requiring special accessibility needs.

Show my current location
Our project is based on the OSM crowdsourcing paradigm – it is an application for everyone to share information about their environment that others can use freely. We do not store any of your data, we only help you to mark and describe the entrances, and submit them to OSM.
The project was inspired by Blindsquare`s need for entrance-location data, and it was built in the scope of the course GIS Application Development in Aalto University by Aditya Raju, Anna Gorodetskaya, Kamyar Hasanzadeh, Marco Martins.
